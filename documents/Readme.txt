Loading up tripRec on your machine

1. install node.js 8.11.3 from https://nodejs.org/en/
2. (windows only) install chocolatey from https://chocolatey.org/
3. on command line: (windows) "choco install meteor" or (Mac/Linux) "curl https://install.meteor.com/ | sh" to install meteor.js
4. use git (https://git-scm.com/downloads) to clone the repo to a folder
    i. git clone https://TripRec@bitbucket.org/triprec_teamjk/server.git
    ii. git clone https://TripRec@bitbucket.org/triprec_teamjk/app.git
5. navigate to folder in command line and type "npm start" or "meteor run"
